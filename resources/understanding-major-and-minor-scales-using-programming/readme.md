# Understanding major and minor scales using programming

## Summary
This script prints out all 15 major and minor scales.

## Prerequisite Knowledge
As this is a standalone script rather than a tutorial, the only pre-requisite knowledge is how to run a Python script.

However, in order to understand the code, basic python knowledge is required.
And to understand its output, basic music notation knowledge is required (letter notation).

## Programming concepts and methods
Programming concepts:
* for loops
* string formatting
* sets
* functions
* default arguments
* documentation strings
* str.count: counting character occurences in a string

Music concepts:
* scales
* major scales
* minor scales
* circle of fifths

## How this resource could be used
It could be used either to teach programmers about music theory or to teach musicians about programming.
